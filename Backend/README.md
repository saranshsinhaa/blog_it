<h2 align="center">Backend</h2>

<h2 align="center">Setup ENV</h2>

```
#  ---  Config.env  ---

NODE_ENV = development
PORT =5000
URI =http://localhost:3000
MONGO_URI =
JWT_SECRET_KEY =
JWT_EXPIRE = 60m
RESET_PASSWORD_EXPIRE = 3600000
```

<h2 align="center">Install Backend dependencies</h2>

```

$ cd Backend
$ npm install
$ npm run start
```
